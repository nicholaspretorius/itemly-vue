### Itemly Todo App in Vue

#### Dependencies

* npm i --save babel babel-preset-env babel-register webpack css-loader html-webpack-plugin vue vue-loader vue-template-compiler webpack-cli
* npm i --save-dev webpack-dev-server
* npm i webpack-cli

#### Installation 

$ npm i 

#### Development Server

Run: 


```webpack-dev-server```

Stop: 

Ctrl + C
